- for: es un bucle que nos permite llevar a cabo una condición.
	Similar al if.
Se declara for (declaramos la variable; expresión condicional; incrementamos la variable){
}
Ej.
for (let i = 0; i < 10; i++){
	console.log (i);
	}

- Y (&&) es un operador lógico el cual evalúa dos expresiones.
	Devuelve verdadero solo si ambas condiciones son verdaderas.
	Si una (o ambas) expresiones son falsas, devuelve falso.

- O (||) es un operador lógico el cual evalúa dos expresiones.
	Determina si una de las dos expresiones son verdaderas.
	Devuelve true si una (o ambas) expresiones son true.
	Devuelve false si AMBAS son false.

- ! Es un operador lógico.
	Devuelve lo opuesto de lo que realmente pasa.
